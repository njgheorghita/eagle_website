# Eagle Bakery

Website for [Eagle Bakery](https://eaglebakerydoco.com) — a seasonal artisan croissant and pastry operation in Ephraim, Door County, Wisconsin.

## Stack

Single `index.html` file. No build step, no dependencies. Hosted on GitHub Pages.

## Deployment

1. Push `index.html` to `main`
2. Settings → Pages → Deploy from `main` branch
3. Add a CNAME record pointing `eaglebakerydoco.com` to `<your-username>.github.io`

## Updating

Just edit `index.html` and push. Things you'll likely update:

- **Hours** — search for `hours-row`
- **Address** — search for `Wilder Inn`
- **Instagram handle** — search for `@eaglebakery`
- **Email** — search for `hello@eaglebakerydoco.com`
